# RootsSky Design System & Brand Identity

This document defines the official design tokens, typography, color palette, and visual identity for **RootsSky** (Premium Minecraft Skyblock Server).

---

## 🎨 Color Palette

We utilize a hybrid theme that fuses organic growth (Roots) with celestial magic (Sky). This is implemented using a sleek, modern dark-mode aesthetic with vibrant green, teal, and gold accents.

### 🌌 Backgrounds & Surfaces (Void & Obsidian)
*   **Void Black (Primary Bg):** `#070A13` — The absolute dark sky backdrop.
*   **Obsidian Dark (Surface Bg):** `#0E1322` — Cards, navigation bar, and container backgrounds.
*   **Slate Border / Outline:** `#1E293B` — Subtle dividing lines.
*   **Glassmorphic Overlay:** `rgba(14, 19, 34, 0.6)` with `backdrop-filter: blur(12px)`.

### 🌿 Primary Colors (Growth & Nature - Roots)
*   **Ethereal Emerald (Default):** `#10B981` (Emerald 500)
*   **Deep Forest (Dark Accent):** `#047857` (Emerald 700)
*   **Mint Glow (Highlight):** `#34D399` (Emerald 400)

### ☄️ Secondary Colors (Celestial & Space - Sky)
*   **Sky Cyan (Default):** `#06B6D4` (Cyan 500)
*   **Deep Ethereal Blue:** `#1D4ED8` (Blue 700)
*   **Celestial Glow:** `#22D3EE` (Cyan 400)

### 👑 Premium Accents (Vibrant Gold & VIP)
*   **Crown Gold:** `#F59E0B` (Amber 500) — Used for premium ranks, store CTAs, and active statuses.
*   **Sunburst Orange:** `#D97706` (Amber 600) — Hover states for premium elements.

---

## ✍️ Typography

To convey both the high-quality modern community feel and the fantasy skyblock theme, we use two primary Google Fonts.

### 1. Headings & Hero Titles
*   **Font Family:** `Outfit`, sans-serif
*   **Weights:** `700` (Bold), `800` (Extra Bold)
*   **Style:** Clean, geometric, high-tech, and futuristic yet premium.

### 2. Body Text & Interface
*   **Font Family:** `Inter`, sans-serif
*   **Weights:** `400` (Regular), `500` (Medium), `600` (Semi-Bold)
*   **Style:** Exceptionally clean, optimized for screen readability and high-density information.

### 3. Server IP & Technical Elements
*   **Font Family:** `JetBrains Mono`, monospace
*   **Weights:** `500` (Medium)
*   **Usage:** IP Address copy widgets, console commands, stats.

---

## 🔘 Interactive Elements & Buttons

### ⚡ Primary Action Button (Play Now / Join)
*   **Background:** Linear Gradient `to right from #10B981 to #06B6D4`
*   **Hover State:** Expand gradient, add soft glow `box-shadow: 0 0 20px rgba(16, 185, 129, 0.4)`
*   **Text:** `#FFFFFF` (Bold, `Outfit`)

### 🛒 Store & Premium CTA
*   **Background:** Linear Gradient `to right from #F59E0B to #D97706`
*   **Hover State:** Glow effect `box-shadow: 0 0 20px rgba(245, 158, 11, 0.4)`
*   **Text:** `#FFFFFF` (Bold, `Outfit`)

### 📄 Secondary / Outline Buttons
*   **Border:** `1px solid #1E293B`
*   **Background:** Transparent / `rgba(255, 255, 255, 0.03)`
*   **Hover State:** Background becomes `rgba(255, 255, 255, 0.08)`, Border color changes to Primary Green.
